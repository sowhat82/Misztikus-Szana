## Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/Misztikus-Szana.git
cd Misztikus-Szana
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the app:
```bash
streamlit run app.py
```

4. Open your browser to `http://localhost:8501`

## How to Use

1. **Welcome Screen**: Choose whether you're ready for a full reading or just want a fortune cookie
2. **Select Topic**: Choose from Health, Wealth, Relationship, or Empowerment
3. **Enter Details**: Provide your name and birth date for personalized readings
4. **Interactive Divination**:

   - Burst the cosmic orb
5. **Receive Reading**: Get your personalized mystical reading combining multiple divination systems

## Reading Generation

Readings utilize the **Barnum Effect** - statements that seem personal but apply broadly. The app combines:
- User's zodiac sign traits
- Chinese zodiac characteristics
- Selected Tarot card meanings
- Dice numerology
- Orb color symbolism
- Topic-specific guidance

Each reading is unique and personalized while maintaining the mystical, vague quality that makes divination engaging.

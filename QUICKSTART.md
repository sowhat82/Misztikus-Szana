## First Time Usage

1. You start with **3 free tokens**
2. Click **"Yes, I'm Ready"** to begin your first reading
3. Choose a topic: Health, Wealth, Relationship, or Empowerment
4. Enter your name and birth date
5. Interact with the mystical elements:
   - Choose a Tarot card

2. **Refer a Friend** - Share your referral link (2 tokens per referral)
3. **Bonus System** - Complete 3 readings in one session = 1 free token!

## Stopping the App

To stop the Streamlit server:
```bash
pkill -f streamlit
```

To restart:
```bash
streamlit run app.py
```

## Troubleshooting

- **App won't load?** Make sure port 8501 is available
- **Tokens not updating?** Refresh the page
- **Styling issues?** Clear your browser cache
